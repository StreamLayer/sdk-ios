How to use this documentation

Introduction

Welcome to the StreamLayer iOS documentation hub where you can find all the information you need to get started. The StreamLayer SDK will enable you to transform the viewing experience for your users by seamlessly integrating interactive features and functionality directly into your app environment.

  1. As a starting point, please read the About StreamLayer section, which will give you a quick overview of the SDK and its capabilities.
  2. We then recommend reading the Architecture Overview section, which will give you a macro overview of the integration process.
  3. From there, the Integration Guide section will walk you through the specific steps involved in successfully completing the integration.
  4. Finally, as an additional resource, we offer the Sample Integrations section where you can find functioning examples of the integrated SDK.

Support

If you have any questions, please do not hesitate to reach out to support@streamlayer.io. If you have already begun an integration with us, please request that we add you to our Slack channel for more immediate communication with our engineering support team.
