Overlay Visibility Handling Guide

The implementation steps required here will depend largely on the interplay of your app’s interface and the SDK interface. In certain cases, you may need to hide certain interface elements and/or resize the video.

Here is a basic example of the delegate methods implementation.

SLRVideoPlayer: SLROverlayDelegate {

...

  public func overlayOpened() {
    print("overlay opened")
    // Put your logic here
  }

  public func overlayClosed() {
    print("overlay closed")
    // Put your logic here
  }

...

}
