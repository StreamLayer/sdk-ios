Overlay Visibility Handling Guide

The operations you have to implement highly depends on the interplay of your app’s interface and the SDK interface. You can hide some of your interface elements, or resize video.

Here is a basic example of the delegate methods implementation.

SLRVideoPlayer: SLROverlayDelegate {

...

  public func overlayOpened() {
    print("overlay opened")
    // Put your logic here
  }

  public func overlayClosed() {
    print("overlay closed")
    // Put your logic here
  }

...

}
