  • changeStreamEvent(for:)

    SDK supports event by event configuration. After StreamLayer SDK has been initialized to configure make sure to call this method at least once to fetch active configuration.

    Every time current event is changing this method should be called with a new event id to fetch relevant configuration

  • requestDemoStreams(showAllStreams:completion:)

    Request demo streams by HostApp cross StreamLayer. Streams of lives games to be displayed in the host app. For demonstration purposes only. Data requested by gRPC bridge.

    • showAllStreams: By default we receive only game events with a stream ulr. But sometimes (e.g. to test statistics data of real events) we need to test some cases with streams without video urls. If showAllStreams - true, will be received all objects showAllStreams - false, only events with ulr streams
    • completion: Callback with remote data/error.
